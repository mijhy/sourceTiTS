showBust

public function showBust(busts: Array):void

Pass in the name of the character bust to display.  Presently, this is using the “old” system of a capitalised string index to find a bust; it should probably be refactored to instead use the shortName variable(s) of the busts chosen for display.  Any index NOT found will hide the bust, ergo “hide”, “none” “somebustthatdoesntexist” will all hide the bust display.  Currently, the RIVAL images won’t be displayed.  I’ll fix it when I’ve rigged everything else up.
